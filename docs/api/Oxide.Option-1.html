    Class Option<TOption>

    The Option<TOption> type represents an optional value. Every option is either a Some<T>, and has a value, or is a None<T> that has no value.

    Name Description
    TOption

    The type of the wrapped value.
    Remarks

    Option types can be useful in a number of places:

    • Replace null or other "no value" markers (initial values, optional parameters, etc.)
    • Return values for partially defined (in the mathematical sense) functions
    • Return values for simple error reporting, where a None<T> can be returned
